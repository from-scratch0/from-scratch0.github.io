文档元数据

内容分区

元素
<article>* 表示一个完整的、自成一体的内容块，比如博客文章、新闻报道、用户评论、论坛帖子
<aside>* 添加附注，可以用来介绍另一个相关话题、解释文档中某个观点；或广告、相关内容链接、醒目引文
<header>*
<h1>–<h6>
<hgroup>* 标注副标题，结构上只关注顶级标题，其它标题会显示在浏览器中，但不会被列入文档纲要
<main>
<footer>*
<nav>* 标注导航链接，目录及索引
<section>* 包含在HTML文档中的独立部分

元素 属性
<div> 通用型的流内容容器
<p>
<figure>* 添加插图，用<figcaption>添加图题或说明
<figcaption>
<blockquote> 引用内容，可以将原内容的出处 URL 地址设置到 cite 特性上，若要以文本的形式告知读者引文的出处时，可以通过 cite 元素 cite
<hr> 主题划分
<pre> 预定义格式文本
<ul> 无序列表
<ol> 有序列表，通常渲染为一个带编号的列表
<li>
<dl> 描述列表，包含术语定义以及描述
<dt>
<dd>

<kbd> 表示用户输入，以浏览器的默认monospace字体显示
<del> 被删除的文字内容
<iframe> 将另一个HTML页面嵌入到当前页面中
